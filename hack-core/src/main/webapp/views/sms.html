Subscribing using SMS

Send SMS to +44 20 3095 4390 with your location only e.g.

London UK
You should get a confirmation when your subscription has been received.

Reporting an incident using SMS

Send SMS to +44 1288 255116 with a location, incident e.g. 

Kew UK, Attack has happened
The formatting of these messages (I.e. the comma in the second one) is currently important TIP: If your location is recognised in the web version edit box it will be recognised on the SMS interface.
